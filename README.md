# Supermarket Sales Analysis & Dashboard 

## Overview
This project focuses on data quality and interactive data visualization for supermarket sales. The goal was to clean raw data, generate a Data Quality Report, and build an interactive dashboard to extract actionable business insights.

## Technologies Used 
* **Data Cleaning & Manipulation:** Python (Pandas)
* **Data Visualization & BI:** Power BI
* **Data Engineering Concepts:** ETL Processes, Data Modeling (Fact & Dimension Tables)

## Project Workflow 
1. **Data Loading:** Extracted the raw supermarket dataset.
2. **Data Cleaning (Pandas):** 
   - Handled missing values and removed duplicates.
   - Standardized data types and corrected formatting errors.
3. **Data Quality Report:** Generated a before-and-after summary table to highlight the data cleaning process.
4. **Data Visualization (Power BI):** 
   - Imported the clean dataset into Power BI.
   - Built an interactive dashboard showing sales trends, product performance, and customer demographics.

## Key Insights & Results 
* Improved data reliability for accurate reporting.
* Visualized key performance indicators (KPIs) to help stakeholders make data-driven decisions.
